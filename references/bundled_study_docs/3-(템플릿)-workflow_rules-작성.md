# 자동화 설계 작성 템플릿

## 이 문서의 역할

이 문서는 `자동화 설계 초안 작성용 템플릿`입니다.

중요:

- 이 문서는 인터뷰 문서가 아닙니다.
- 이 문서는 최종 결과물을 대신하지 않습니다.
- 이 문서는 `실행 가능한 초안`을 빠르게 만드는 참고 템플릿입니다.

## 언제 사용하는가

- 워크시트를 정리한 직후 첫 자동화 설계 초안을 만들 때
- 도구 연결 전에 처리 흐름을 먼저 정리하고 싶을 때

## 먼저 준비할 입력

| 준비물 | 최소 요구 |
|---|---|
| 업무 한 줄 설명 | 반드시 필요 |
| 시작 조건 | 반드시 필요 |
| 입력 예시 | 1건 이상 |
| 출력 예시 | 1건 이상 |
| 현재 가능한 처리 방식 | 1건 이상 |
| 예외 사례 | 1건 이상 |

가능하면 함께 주면 좋은 자료:

- 인터뷰 답변 메모
- 워크시트 초안
- 관련 예시 파일

## 반드시 포함해야 하는 항목

1. 목적
2. 시작 트리거
3. 입력 수집과 검증
4. 핵심 처리 단계
5. 사람 검토 지점
6. 저장과 전달
7. 예외 처리
8. 중복 방지
9. 현재 가능한 자동화 범위
10. `[확인 필요]` 항목
