# 중급 자동화 실행 허브

## 이 문서의 역할

이 문서는 현재 중급 자동화 스킬의 `입구 문서`입니다.

이 문서의 목적:

1. 어떤 문서를 어떤 순서로 참고해야 하는지 알려줍니다.
2. 각 문서가 어떤 역할을 맡는지 구분해 줍니다.
3. 어떤 산출물을 남겨야 하는지 알려줍니다.
4. 사용자가 복잡한 내부 체계를 몰라도 흐름을 따라갈 수 있게 합니다.

## 먼저 구분할 것

### 참고 문서 위치

```text
<skill_root>\references\bundled_study_docs
```

### 실습 산출물 권장 위치

```text
<project_root>\
  source\
  output\
```

핵심 원칙:

- 참고 문서는 읽는 용도입니다.
- 결과물은 `output`에 따로 둡니다.
- 작업 중인 자료는 `source`에 둡니다.
- 복잡한 상태 폴더는 만들지 않습니다.

## 순차 실행 맵

### 1. 인터뷰

- 문서: `1-업무정의-자동화-위임설계서-인터뷰.md`

- 산출물: 인터뷰 메모
- 규칙: 질문은 한 번에 1개씩 진행하고, 사용자가 현재 답할 수 없는 항목만 `[보류: 이유]`
