지금 업무를 중급 자동화 6단계 흐름으로 진행해줘.
인터뷰 -> 워크시트 -> 자동화 설계 -> 도구 연결 -> 오케스트레이션 실행 및 결과 확인 -> Agent 변환
순서로 진행하고, 복잡한 폴더 구조나 상태관리는 만들지 말아줘.
각 단계에 들어가기 전에는 `python .claude\skills\_education_skills_중급\automation-pipeline-design-eduv-중급\scripts\find_gpters_study_docs.py --stage <현재단계>`로 현재 번들 문서 매핑을 다시 확인하고, 번호가 아니라 역할 매핑 결과를 우선 사용해줘.
`<현재단계>`에는 `intent`, `interview`, `worksheet`, `automation_design`, `tool_connection`, `orchestration_execution`, `agent_conversion` 중 하나를 사용해줘.
특히 인터뷰와 워크시트를 가장 중요하게 다뤄서, 인터뷰가 충분히 끝나기 전에는 다음 단계로 넘어가지 말아줘.
현재 작업공간에서 실제로 찾은 SKILL.md를 먼저 검사해서 도구 활용 가능성을 검토하고,
없는 기능은 그레이 영역으로 제안해줘.
인터뷰 단계에서는 질문을 한 번에 1개씩만 하고, 사용자가 더이상 답할 수 없는 항목만 [보류: 이유]로 남겨줘.
워크시트 핵심 섹션이 채워지기 전에는 자동화 설계로 넘어가지 말아줘.
오케스트레이션 실행 및 결과 확인이 끝나면 `agent_conversion` 단계에서 Agent 정의 파일까지 정리해줘.
